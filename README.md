# 家計簿ウェブアプリ

日々の収支を管理するためのウェブベースの家計簿ツールです。

## 機能

- **カレンダービュー**: 月単位のカレンダーを表示し、日付をクリックして収支を記録できます
- **収支入力モーダル**: 収入・支出を切り替えて、カテゴリ、金額、メモを入力できます
- **月別サマリーテーブル**: 年間の月別×カテゴリ別の収入・支出合計を一覧表で確認できます
- **データ永続化**: 入力した収支データはブラウザの localStorage に自動保存されます

## 収支区分

- **支出**: 生活用品・コロナ用品・魚・サブスク・その他
- **収入**: 給与・事業収入・副業収入・返金・その他

