# 家計簿ウェブアプリ

日々の支出を管理するためのウェブベースの家計簿ツールです。

## 機能

- **カレンダービュー**: 月単位のカレンダーを表示し、日付をクリックして支出を記録できます
- **支出入力モーダル**: カテゴリ（食費・交通費・住居費・光熱費・娯楽費・医療費・衣服費・その他）、金額、メモを入力できます
- **月別サマリーテーブル**: 年間の月別×カテゴリ別支出合計を一覧表で確認できます
- **データ永続化**: 入力した支出データはブラウザの localStorage に自動保存されます

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
npm run preview
```
